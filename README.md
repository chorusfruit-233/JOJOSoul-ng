# JOJO Soul (Refactored)

这是一个基于 Python 的 JOJO 奇妙冒险同人游戏，经过了重构和优化。

## 游戏简介

你降落在一个被普奇神父控制的大陆上，他想重启世界。你需要通过战斗和升级来阻止他，拯救这个世界！

## 游戏特色

- 🎭 基于 JOJO 奇妙冒险的原创剧情
- ⚔️ 五种元素属性的战斗系统（火焰、水、土地、暗黑魔法、闪光）
- 🗺️ 9个不同的地图区域，包括4个新增的高级区域
- 👥 8种不同类型的敌人，每个都有独特的元素抗性
- 🛒 扩展的商店系统，新增5种高级装备和道具
- 🎯 技能系统，5个主动技能带来更多战斗策略
- 🏆 成就系统，15个成就等你完成
- 🎯 五种难度选择（无限金币版、简单、普通、坤难、炼狱）
- 🎮 简单易用的图形界面
- 🌟 隐藏结局和彩蛋系统
- 💾 完整的存档系统，支持所有新功能

## 快速开始

### 环境要求

- Python 3.12+
- 支持 GUI 的操作系统环境

### 安装与运行

1. **克隆仓库**
   ```bash
   git clone git@github.com:chorusfruit-233/JOJOSoul-ng.git
   cd JOJOSoul-ng
   ```

2. **创建虚拟环境**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Linux/Mac
   # 或
   venv\Scripts\activate     # Windows
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **运行游戏**
   ```bash
   python JOJOSoul-ng.py
   ```

### 打包为可执行文件

```bash
pyinstaller --onefile --windowed --name "JOJOSoul" JOJOSoul-ng.py
```

## 游戏玩法

### 角色属性

- **生命值**: 角色的生存能力
- **攻击力**: 基础伤害数值
- **金币**: 购买装备和道具的货币
- **伤害倍率**: 暴击伤害的范围（0x-2x）
- **元素伤害加成**: 装备带来的元素伤害提升
- **技能点**: 升级获得，用于学习和升级技能
- **纯氧数量**: 特殊道具，用于解锁隐藏结局

### 战斗系统

每个敌人对不同元素有不同的抗性：
- 🔥 **火焰**: 对植物类敌人有效
- 💧 **水**: 对火属性敌人有效
- 🌍 **土地**: 对水属性敌人有效
- 🌑 **暗黑魔法**: 高伤害元素
- ✨ **闪光**: 特殊效果，对某些敌人有奇效

### 地图区域

- 🌳 **丛林**: 对战树妖
- ⛰️ **山洞**: 对战吸血鬼
- 🦠 **腐化之地**: 对战沼泽怪
- 🌋 **熔岩地下城**: 对战熔岩怪
- 🏔️ **雪山**: 对战冰霜巨人（5级解锁）
- 🌑 **暗影遗迹**: 对战暗影刺客（击败沼泽怪解锁）
- ⛈️ **风暴高地**: 对战雷电元素（击败熔岩怪解锁）
- 🏛️ **古代神殿**: 对战高级敌人（8级解锁）
- 🌌 **天国**: 最终 Boss 战

### 商店物品

#### 基础装备
- 🛡️ **盔甲** [100G]: +30 生命上限
- ⚔️ **剑** [100G]: +5 攻击力
- 🧪 **药水** [50G]: 回满生命值
- 📦 **宝箱** [70G]: 随机属性变化

#### 高级装备（3级解锁）
- 🧙 **魔法袍** [200G]: +15% 元素伤害
- 💪 **力量护符** [150G]: +3 伤害下限倍率
- 🛡️ **守护盾** [180G]: +50 生命上限
- ⭐ **经验药水** [80G]: +100 经验值

#### 特殊道具（5级解锁）
- 📜 **元素卷轴** [120G]: 临时增强某元素伤害2倍，持续3回合

### 技能系统

升级获得技能点，可以学习和升级以下技能：

- 🔥 **火球术**: 造成50点火焰伤害，无视元素倍率
- 💚 **治疗术**: 恢复50%最大生命值
- 🛡️ **护盾**: 下回合受到伤害减半
- 💥 **元素爆发**: 所有元素伤害倍率x2，持续3回合
- ⏰ **时间减缓**: 敌人下回合无法攻击

每个技能都有独立的冷却时间，合理使用技能是战斗胜利的关键！

### 成就系统

完成各种挑战，解锁成就并获得金币奖励：

#### 战斗成就
- **初次胜利**: 击败第一个敌人
- **小有成就**: 击败10个敌人
- **怪物猎人**: 击败50个敌人

#### 收集成就
- **富有之人**: 拥有1000金币
- **大富翁**: 拥有10000金币

#### 成长成就
- **等级大师**: 达到10级
- **传奇勇者**: 达到20级

#### 特殊成就
- **元素大师**: 购买魔法袍
- **技能新手**: 学习第一个技能
- **技能大师**: 所有技能达到3级
- **雪山征服者**: 击败冰霜巨人
- **暗影克星**: 击败暗影刺客
- **风暴掌控者**: 击败雷电元素
- **神殿英雄**: 击败古代神殿所有敌人
- **世界拯救者**: 击败普奇神父
